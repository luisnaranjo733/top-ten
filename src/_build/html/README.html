Project Structure¶

Index.py¶

This is the website’s ‘homepage’, the index.

From here you access all ten events.

event/¶

This folder contains template.py which creates the base structure of the project.

It also contains 10 restructured text files (created by template.py).

Each file corresponds to one of the 10 most important events in U.S. history since the 1900’s.
